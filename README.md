# dw

- It's an Xhamster video downloader.
- If given a URL, it generates option of available video resolutions.
- It tries the file path with an array of base URLs if it gets an error.
- If provided a channel/creator url/name, it provides list of all videos present with their resolutions and length. Every video has a checkbox which indicates if it should be downloaded, user selects the download resolution here as well. This is batch download, one video is downloaded after another.
- If provided a list of URLs, it parsed them and provides download options for each.


Note
- Every video parsed for download contains available resolutions and timestamp.
